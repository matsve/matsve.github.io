This is the home page of matsve. I am a programmer that focuses on user experience, both from a developer perspective and a user perspective. Libraries should be powerful but still simple to use and user interfaces should be visually appealing and easy to use while still putting the full power in the hands of the user.

Two of my projects that is worth checking out is Ini and CSL, both available as open source-projects on Github (links to the left). Ini is written in C# and is a minimalistic approach to an ini-file handling library. It's super easy to make use of but at the same time quite powerful.
CSL (Custom Script Language) is the same, but for scripts instead of ini files. There are a quick example of how it works on the project page on Github, and it is implemented for both C++ and C#.
